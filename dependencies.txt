Si vas a montar exactamente el mismo proyecto en otro PC (Python 3.12, OpenCV, reconocimiento facial y PostgreSQL), estos son los comandos que utilizaste.

## 1. Actualizar pip

```powershell
py -m pip install --upgrade pip
```
## ?. python-dotenv

```powershell

py -m pip install python-dotenv

py -m pip install PySide6

---

## 2. Instalar setuptools compatible

(Importante porque `face_recognition_models` usa `pkg_resources`)

```powershell
py -m pip install setuptools==80.9.0
```

---

## 3. Instalar NumPy compatible con dlib

```powershell
py -m pip install numpy==1.26.4
```

---

## 4. Instalar OpenCV compatible

La versión nueva pide NumPy 2.x, así que instala una versión compatible:

```powershell
py -m pip install opencv-python==4.9.0.80
```

---

## 5. Instalar dlib

Descarga primero el archivo:

```
dlib-19.24.99-cp312-cp312-win_amd64.whl
```

y luego:

```powershell
cd C:\Users\TU_USUARIO\Downloads

py -m pip install dlib-19.24.99-cp312-cp312-win_amd64.whl
```

---

## 6. Instalar face_recognition

```powershell
py -m pip install face-recognition
```

---

## 7. Instalar PostgreSQL para Python

```powershell
py -m pip install psycopg2-binary
```

---

## 8. Verificar las instalaciones

```powershell
python -c "import cv2; print(cv2.__version__)"

python -c "import numpy; print(numpy.__version__)"

python -c "import dlib; print(dlib.__version__)"

python -c "import face_recognition; print('OK')"

python -c "import psycopg2; print('OK')"
```

---

## Instalación completa en un solo bloque

```powershell
python -m pip install --upgrade pip

pip install setuptools==80.9.0

pip install numpy==1.26.4

pip install opencv-python==4.9.0.80

pip install face-recognition

pip install psycopg2-binary
```

Después instala manualmente:

```powershell
python -m pip install dlib-19.24.99-cp312-cp312-win_amd64.whl
```

---

También te recomendaría crear un archivo `requirements.txt` para tu proyecto, de forma que en cualquier PC solo tengas que ejecutar:

```powershell
pip install -r requirements.txt
```

y se instale todo automáticamente con las mismas versiones.
